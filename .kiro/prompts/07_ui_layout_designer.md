# Role: Senior UI/UX Designer & Frontend Engineer
## Task
Design the NimbusAI interface using a "Design Engineering" approach.

## Key Features to Implement
- **Infinite Canvas**: Central area powered by React Flow for architecture visualization.
- **BYO-API Dashboard**: A glassmorphism panel for users to enter AWS keys and external API credentials.
- **Floating Cost HUD**: Real-time dollar estimate that updates based on canvas changes.
- **Top Actions**: "Generate", "Export Terraform", and "Security Audit" buttons.

## Visual Style: "Nimbus Premium"
- **Background**: #050510 with a subtle dotted grid.
- **Surfaces**: Glassmorphism (backdrop-blur: 24px, border: 1px solid rgba(255,255,255,0.1)).
- **Primary Accent**: Gold #E8C77C (used for primary actions and highlights).
- **Secondary Accent**: Neon Violet (used for data flow animations).

## Output Requirement
Provide a complete component hierarchy and Tailwind utility mapping for the layout. Include Framer Motion transition definitions for a "fluid" feel.
